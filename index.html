Why Tlumx?

  • Performance - Tlumx framework is very fast and outperforms most of its competitors
  • Flexible and Extensible - allows you to easily use third-party packages or expand existing
  • Secure - helps you write secure code
  • Engineered with best practice in mind - Tlumx framework use MVC and Middleware architecture, supports dependency injection, event manager, fast HTTP router and many other
  • Standardization - following PHP-FIG standards
  • Easy to use - which will allow beginners to start working with it very quickly
  • Small size - packages contain all the necessary functionality with a small size
  • Available on GitHub - Tlumx framework, packages and other solutions are available on GitHub and can be installed via Composer dependency manager
  • MIT license - FREE and open-source licensed
  • Suitable for writing applications of any complexity
